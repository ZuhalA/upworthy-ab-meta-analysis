# The Science of Clicks: A Meta-Analysis of Real A/B Tests

The statistical analysis of 4,873 headline A/B tests conducted by Upworthy (2013-2015), exploring what drives click-through rates, how often A/B tests find real effects, and the statistical pitfalls of online experimentation.

Dataset: Upworthy Research Archive (Matias et al., 2021, Nature Scientific Data) — 22,666 headline packages, 538M+ participant assignments.
